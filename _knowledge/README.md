# Knowledge Base Hub

This folder is the reference-knowledge counterpart to backlog delivery tracking.

Use this area for curated knowledge sources (books, papers, wiki packs, deep notes) decomposed into topic folders for rapid non-linear learning.

## Principles
1. One source folder per core knowledge artifact.
2. Decompose into topic nodes you can learn and query independently.
3. Preserve stable IDs for nodes and links.
4. Prefer concise summaries and snippets over chapter-by-chapter transcription.
5. Update metadata only when adding meaningful knowledge or reorganizing topology.

## Source Types
- Books
- Research papers
- Wiki collections
- Internal deep-dive notes
